# DOMAIN_POLICY.md

# Philosophy

Domains define:
- optimization goals
- interpretation style
- risk sensitivity
- guidance behavior

Domains NEVER alter:
- astronomy
- features
- symbolic states

Domains are read-only interpreters.

---

# 1. GENERAL_LIFE DOMAIN

## Purpose
Provide balanced life guidance for normal users.

## Optimizes For
- stability
- adaptability
- growth
- wellbeing
- resilience

## Interpretation Style
Neutral, psychologically safe, non-extreme.

# 7. DOMAIN EXECUTION MODEL

```text
Astronomy
→ Features
→ Symbolic States
→ Event Activation
→ Domain Interpretation
→ Decisions
```

Domains:
- consume symbolic states
- apply domain-specific meaning
- generate guidance
- never modify lower layers
