# Wealth, Financial Gain & Accumulation — 5-Layer Architecture

## Domain

Finance → Wealth accumulation, monetary gain, asset growth, speculative success.

## Chart Model

**Single-chart.** This domain evaluates one person's natal chart for wealth potential and timing.

## Key Significators

| Layer | Planets | Houses |
|-------|---------|--------|
| Dasha/Transit | Jupiter, Venus, Mercury, Rahu | 2, 4, 8, 9, 10, 11 |
| Fast Trigger | Moon, Mars, Mercury | Nakshatra-based (Anga Gochara) |
| Classical | 2nd lord, 11th lord, 5th lord, 8th lord, 9th lord | 1, 2, 5, 8, 9, 10, 11 |
| Outcome | Moon, Mars, Venus, Jupiter, Saturn | 5, 6, 8, 11, 12 |

## Evaluation Flow

```
Dasha gate (Jupiter/Venus/Mercury/Rahu in wealth houses from Moon)
  → Transit activation (slow planet confirms window)
    → Fast trigger (Moon/Mars/Mercury Anga Gochara pinpoints day)
      → Classical modifier (Dhana Yogas adjust confidence)
        → Outcome classification (wealth type: stable/speculative/hidden/network)
```
